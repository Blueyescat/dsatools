DSA Tools

A web app for various tools for the Deep Space Airships game.

  • Its files are cached in the browser, so they can be used without needing to download them each time. This means you don't need an internet connection to open and use the cached version of the app, except for some online features.
  • It can be installed on the device for quick access. Most browsers, excluding Firefox and Opera, for example, support this. Your main browser doesn't really matter, just install using any browser. On desktop browsers, you should see a button on the right side of your address bar to install. On mobile browsers, it may be called "add to home screen". For Safari on iOS, you need to launch the installed app once after adding it to home screen.

Tools

Blueprint Editor

Various blueprint editing tools: Crop, Flip, Rotate, and Replace Items.

Blueprint Bin

WIP blueprint storing and sharing service.

Image to Pixel Art

Converts an image into in-game pixel art and shows which color to place where, based on the closest paint color for each pixel, along with coordinates on the ship.

Item List

Small tool that nicely lists items in the game.
